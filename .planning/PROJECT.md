# Project: Hệ thống Quản lý KTX

## Vision
Xây dựng hệ thống quản lý ký túc xá thông minh, bảo mật cao, đạt tiêu chuẩn SaaS Enterprise cho các trường đại học/cao đẳng (Ví dụ: KTX Phương Đông).

## Core Values
- **Security First:** Bảo mật PII sinh viên bằng Encryption và Blind Indexing.
- **Atomic Operations:** Chống Race Condition trong việc xếp phòng.
- **Linear Aesthetics:** Giao diện tối giản, chuyên nghiệp, chuẩn editorial.

## Tech Stack
- **Backend:** Laravel 11.x, PHP 8.2+.
- **Frontend:** Blade, Tailwind CSS, Impeccable UI Components.
- **Tools:** GSD (Task Management), GitNexus (Code Intel), Vibe-cn (Methodology).
